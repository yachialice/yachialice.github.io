LeetCode-215. Kth Largest Element in an Array

Description

Given an integer array nums and an integer k, return the kth largest element in the array.

Note that it is the kth largest element in the sorted order, not the kth distinct element.

Constraints:

  • 1 <= k <= nums.length <= 104
  • -104 <= nums[i] <= 104

Solution

直接从大到小排序，取第k个。
（1）对于快排来说，swap(nums[right], nums[start])后，以right指针指向的位置为边界，right左侧都小于nums[right]，右侧都大于nums[right]，故而此时nums[right]位置正确。
（2）若从大到小进行快排。则每次nums[right]在正确排序位置，nums[right]为第k+1大的元素。
若right == k-1，找到第k大元素。若right < k-1，证明第k大元素在right右侧，从而start = right+1；若right > k-1，证明第k大元素在right左侧，end = right-1;
