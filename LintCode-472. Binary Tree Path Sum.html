LintCode-472. Binary Tree Path Sum |||

Description

给一棵二叉树和一个目标值，找到二叉树上所有的和为该目标值的路径。路径可以从二叉树的任意节点出发，任意节点结束。

Solution

（1）先序遍历，将点存到node_list中

（2）node_list中每一个点为起点，寻找路径

（3）因为ParentTreeNode中有parent指针，因此dfs时可以向parent方向遍历。

（4）为了防止路径走反，走重复，或者kid->parent，parent->kid打转，引入一个set记录已经遍历过的节点。这个set在dfs前初始化为空，在dfs过程中只insert，不erase。我们保证dfs遍历的时候，每条路径都以root为起点，因为是棵树，所以每个节点遍历一次，就能遍历到所有路径，set不需要erase。
