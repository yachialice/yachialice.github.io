LintCode-246. Binary Tree Path Sum||

Description

给一棵二叉树和一个目标值，设计一个算法找到二叉树上的和为该目标值的所有路径。路径可以从任何节点出发和结束，但是需要是一条一直往下走的路线。也就是说，路径上的节点的层级是逐个递增的。

（1）关于可以从任何起点开始这一点，需要递归地遍历binaryTreePathSum2，注意以下：

mid和left、right遍历的并非同一个函数。如果写成left=binaryTreePathSum(root->left, target);,则root->left的左右子树并没有再以左右子树中的点为起点递归下去，那么res中会缺少元素。

（2）关于可以从任何一点结束这一点，不用!root->left&&!root->right即可return，即去掉是叶节点的限制条件。

（3）注意，dfs函数中curr也可以传reference类型，但要注意在返回前需要curr.pop_back()。
